## Annuité anticipatif
- Jusqu'à présent nous avons considérez le paiement d'annuité en fin de période,
- Si maintenant on verse l'annuité anticipativement (début de la période)

### Exo 
- Un leasing pour une voiture de 12.500€, 
- en 3 ans, mensualités anticipatives, 
- taux annuel effectif fr 9%, 

- Calculez le montants des factures mensuelles de location.

- Co  = 12500
- m   = 12
- t   = 36 = 12 * 3
- i_e = 9% (i_effectif annuel)

```
i_mensuel 
TAUX.NOMINAL(9%;12)/12 = 0,72073%

annuité (location mensuelle = VPM net)
=VPM(0,72073%; 36;-12500;0;1) = 392.63
** mettre 1 dans le paramètre type => versement anticipatif **

annuité avec TVA
392.63 * 21% = 82.45 
392.63 + 82.45 = 475.08
ou
392.63 * 1.21 = 475.08

on fige le montant de l'annuité $ = 392.63

ligne 0 juste 12500

ligne 1 
mensualité 392.63
capital remboursé 392.63
SRD 12500 - 392.63 = 12107.37

ligne 2
mensualité 392.63
interet 12107.37 * 0,72073% = 87.26
capital remboursé 392,63 - 87,26 = 305.37
SRD 12107,37 = 305.37 = 11802.01

```
