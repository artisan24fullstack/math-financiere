## Taux effectif ou taux équivalent ou taux réel

- C'est le taux effectivement (ou réellement) payé sur base d'un an 
- lorsque les intérêts sont capitalisés sur une période différente à un an. 
- On cherche dès lors un taux annuel équivalent


```
i_effectif = ( 1 + j(m)/m ) ^ m - 1 
```
#### Exo 2
- quel est le taux effectif annuel correspondant au taux nominal de 2,99% (sur base d'une capitalisation mensuelle)
- j(m)   = 2.99%
- m      = 12 
- j(m)/m = j(12/12) = taux periodique mensuel
     
```
j(m)/m = 2.99%/12 = 0.0024916 = 24.92%    

(1 + 0.2492%) ^ 12 -1 = 0.030317 = 3.03%
```

#### Exo 3
- quel est le taux effectif annuel correspondant au taux nominal de 9,25% 
- (sur base d'une capitalisation trimestrielle)


- j(m)   = 9.25%
- m      = 4 
     
```
(1 + (9.25%/4) ) ^ 4 - 1 = 0.95758 = 9.58%  
```
