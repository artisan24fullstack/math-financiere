# Formules Intérêts simples

## Explication Intérêt
- Ct = valeur acquise
- Co = Capital prêtée
- t = durée du prêt
  
```
Ct = Co + Intérêts
```

### Intérêts simples
- i = taux d'intérêt
- Ct = valeur acquise (valeur future)
- Co = capital de départ (valeur actuelle)
- t = durée
- I = intérêts

```
Ct = Co * ( 1 + i * t)

I = Co * i * t

Ct = Co + I
```

#### Durée exprimée en période

| Périodes (t) | Formules (I)       | 
| :----------- |:------------------:|
| années       | (Co * i * t) / 1   |  
| trimestre    | (Co * i * t) / 4   |  
| mois         | (Co * i * t) / 12  |  
| quinzaines   | (Co * i * t) / 24  |
| jours        | (Co * i * t) / 365 |
