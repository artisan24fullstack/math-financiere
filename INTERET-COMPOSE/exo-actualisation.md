## Exo 1.       
- Quelle est la valeur actuelle de 100.000€ exigibles dans 20 ans, compte tenu d’un taux intérêt de 8% l’an ?

- t = 20
- i = 8%
- Ct = 100000

- Co = ?

```
Co = Ct / (1 + i) ^ t

Co = 100000 / ( 1 + 8%) ^ 20 = 21454.82
```
## Exo 2 
- Une somme de 5 000 € vous sera remise dans 5 ans. 
- Sachant que le taux de placement est de 7 %.
- Quelle est la valeur de ce capital aujourd’hui ?

- t = 5
- i = 7%
- Ct = 5000
  
```
Co = Ct / (1 + i) ^ t

Co = 5000 / (1 + 7%) ^ 5 = 3564.93
```
